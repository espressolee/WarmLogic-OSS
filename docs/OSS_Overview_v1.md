# WarmLogic-OSS Overview v1

Purpose: provide a public, runnable subset of WarmLogic for demos and reproducibility (osctl toy runtime + minimal console).

Included:
- osctl CLI (run/replay/verify) ported from WarmLogic.
- Toy OS v2 workload (synthetic event log, config, JSON schemas).
- Minimal console (Flask + static UI).
- Quickstart docs.

Excluded:
- Real pilot data (Team A/B), internal CE ledger, drift experiments.
- Large paper artifacts and internal notebooks.

See `README.md` and `docs/Quickstart_OSCTL_v1.md` for usage.
